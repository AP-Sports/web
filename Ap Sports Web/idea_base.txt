Idea para la página web de AP Sports

Objetivo

Crear una página web moderna que funcione como catálogo de productos y
genere consultas por WhatsApp. La venta no se realiza desde la web, sino
que el objetivo es que el cliente vea ejemplos de trabajos, conozca la
calidad de los productos y solicite un presupuesto personalizado.

Inicio

-   Imagen principal de colchonetas deportivas.
-   Frase destacada: “Fabricamos colchonetas deportivas a medida.”
-   Botón “Ver productos”.
-   Botón “Solicitar presupuesto”.
-   Botón flotante de WhatsApp siempre visible.

Catálogo

Se mostrarán productos de ejemplo, como: - Colchonetas para gimnasios. -
Colchonetas para artes marciales. - Colchonetas escolares. - Colchonetas
de alta densidad. - Colchonetas plegables.

Cada producto incluirá: - Foto principal y galería de imágenes. -
Descripción. - Materiales. - Espesores disponibles. - Colores
disponibles. - Medidas de ejemplo. - Botón “Consultar por WhatsApp”.

Se aclarará que todos los productos son fabricados a medida y que el
presupuesto se solicita por WhatsApp.

Página del producto

Cada producto tendrá una página con: - Galería de fotos. - Descripción
completa. - Material. - Espesores. - Colores. - Medidas de ejemplo. -
Preguntas y respuestas (estilo Mercado Libre). - Botón “Consultar por
WhatsApp”.

El botón abrirá WhatsApp con un mensaje automático indicando el producto
consultado.

Trabajos realizados

Galería de proyectos terminados para demostrar la calidad del trabajo y
generar confianza.

Preguntas frecuentes

Sección con respuestas a consultas comunes: - ¿Hacen envíos? - ¿Cuánto
demora la fabricación? - ¿Qué densidad de espuma utilizan? - ¿Se pueden
hacer medidas personalizadas? - ¿Qué colores hay disponibles? - ¿Cuáles
son los medios de pago?

Contacto

-   WhatsApp.
-   Horarios de atención.
-   Ubicación (si corresponde).
-   Redes sociales.

WhatsApp flotante

Estará visible en todas las páginas para que el cliente pueda
comunicarse en cualquier momento.

Objetivo final

La página será un catálogo profesional que muestre la calidad de los
productos y facilite que los clientes pidan un presupuesto personalizado
por WhatsApp de forma rápida y sencilla.
